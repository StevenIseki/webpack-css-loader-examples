Redux Deku Example
=====================

* redux, webpack, deku, router5
* styling with css-modules

extended from thomas roch's awesome [router5 examples](https://github.com/router5/examples)

## Run Dev

```
npm install
npm start
open http://localhost:3000
```

## Run Prod

```
npm install
npm run build
npm start:prod
open http://localhost:3000
```
